These files contains all the code needed to map and filter the raw sequencing reads from my Master Thesis.
The configuration file contains directories leeding to the raw fasta files, input for sequencing parameters, and instructions on where to place output files. 
The files workflow.py and the configuration file should be placed in one folder, together with a folder called 'workflow_source' containing the workflow_source.py file and the workflow_templates.py file.
